廣島・岡山・香川之旅

📅 2026.05.17 — 05.23 (七日自由行)

💡 策略：單買票券 + IC卡 (比週遊券省下約 ¥9,600)

📅 5/17（日）廣島：和平巡禮

🕘 12:30 飯店寄放行李後開始

🟢 交通動線

  • 機場巴士：廣島機場 → 廣島車站/巴士中心 (¥1,450 / 55分)
  • 市區移動：廣島路面電車 (廣電) (刷 IC 卡)

🔵 景點安排

  • 和平紀念公園、原爆圓頂屋
  • 和平紀念資料館 (建議預留 2 小時)
  • 本通商店街散策

🔴 美食推薦

  • 晚餐：廣島燒 (推薦：麗ちゃん 或 みっちゃん)

🌧️ 雨備方案

  • 廣島紙鶴塔 (室內觀景)、SOGO/新天地室內商店街。

📅 5/18（一）吳市：海上歷史

🕘 09:00 廣島站出發

🟢 交通動線

  • JR 吳線快速 (安藝路 Liner)：廣島 → 吳 (刷 IC 卡單程 ¥510)

🔵 景點安排

  • 大和博物館 (1/10 戰艦模型)
  • 鐵鯨魚館 (海上自衛隊史料館)
  • 入船山紀念館 / 烏小島公園 (看潛水艇)

🔴 美食推薦

  • 午餐：吳海自咖哩 (車站附近或博物館旁)
  • 點心：珍來軒 吳冷麵 (在地名物)

🌧️ 雨備方案

  • 大和博物館與鐵鯨魚館皆為全室內，受天氣影響小。取消烏小島公園戶外行程。

📅 5/19（二）宮島：嚴島神社

🌊 滿潮 10:52 / 干潮 16:35

🟢 交通動線

  • JR 山陽本線：廣島 → 宮島口 (¥420)
  • 渡輪：宮島口 → 宮島 (¥200 + 入島稅 ¥100，可刷 IC 卡)

🔵 景點安排

  • 11:00 前後：滿潮參拜神社，拍海上鳥居。
  • 千疊閣、五重塔、紅葉谷公園。
  • 16:30 後：干潮步行至大鳥居下方。

🔴 美食推薦

  • 牡蠣屋 Kakiya 或 燒牡蠣攤位。
  • 紅葉饅頭 (推薦炸紅葉饅頭)、穴子魚飯。

🌧️ 雨備方案

  • 嚴島神社迴廊有遮雨，大聖院室內空間多；可縮減紅葉谷公園步道行程。

📅 5/20（三）廣島 → 岡山

🚄 預計搭乘 09:30 新幹線

🟢 交通動線

  • 新幹線：廣島 → 岡山 (單買自由席約 ¥5,610，40分)
  • 路面電車：岡山站前 → 城下站 (刷 IC 卡)

🔵 景點安排

  • 岡山後樂園 (日本三名園)
  • 岡山城 (烏城)
  • 岡山 AEON Mall (全日本最大級) 購物。

🔴 美食推薦

  • 晚餐：ひとり鍋恵 (人氣壽喜燒/火鍋)
  • 岡山水果：白桃或晴王麝香葡萄甜點。

🌧️ 雨備方案

  • 後樂園風景受限時，增加岡山縣立美術館或專攻岡山車站旁 AEON Mall。

📅 5/21（四）倉敷與鷲羽山夕陽

🚗 14:00 倉敷站前取車自駕

🟢 交通動線

  • JR 山陽本線：岡山 → 倉敷 (¥330)
  • 自駕開始：倉敷 → 兒島 → 鷲羽山 → 岡山。

🔵 景點安排

  • 倉敷美觀地區 (白壁土藏建築群)
  • 兒島牛仔褲大道、舊野崎家宅邸。
  • 鷲羽山展望台：瀨戶大橋夕陽絕景 (日落約 19:10)。

🔴 美食推薦

  • 倉敷：有鄰庵 (生雞蛋拌飯/微笑布丁)。
  • 兒島：與島停車場 (瀨戶內海景觀餐廳)。

🌧️ 雨備方案

  • 大原美術館 (倉敷美觀區內)、倉敷 Ivy Square。取消鷲羽山夕陽。

📅 5/22（五）吉備路 → 高松

🚗 12:30 岡山站前還車

🟢 交通動線

  • 自駕：岡山 → 吉備路 → 岡山還車。
  • JR 快速 Marine Liner：岡山 → 高松 (¥1,660，跨越瀨戶大橋)

🔵 景點安排

  • 吉備津神社 (400公尺絕美木製長廊)
  • 吉備津彥神社 (桃太郎故事起源)。
  • 傍晚：高松港散步、高松地標塔看夜景。

🔴 美食推薦

  • 午餐：岡山車站備前拉麵。
  • 晚餐：高松骨付鳥 (推薦：一鶴 Ikkaku)。

🌧️ 雨備方案

  • 高松站旁擁有全日本最長的「全覆蓋式」商店街，適合雨天逛街吃美食。

📅 5/23（六）高松名園・返程

✈️ 19:05 高松機場起飛

🟢 交通動線

  • 琴電：高松築港 → 栗林公園 (刷 IC 卡)
  • 利木津巴士：高松車站 → 高松機場 (¥1,000 / 45分)

🔵 景點安排

  • 栗林公園：特別名勝，建議搭乘南湖周遊船。
  • 北濱 Alley：高松港舊倉庫改建的文青店鋪群。

🔴 美食推薦

  • 讚岐烏龍麵 (推薦：手打十段、竹清 或 栗林公園內上等黑松)。

🌧️ 雨備方案

  • 栗林公園內茶屋休憩；高松機場內也有許多商店與烏龍麵店可最後採買。
